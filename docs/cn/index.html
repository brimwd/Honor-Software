我们对新冠病毒肺炎 (COVID-19)的回应

COVID-19公告

荣耀软件致力于投资于继续教育并回馈社区。

通过雇用我们通常在内部处理的任务，Honor为因COVID-19而失去收入的个人创造了临时工作和机会来代替收入。

此外，Honor感到自豪的是在此困难时期继续支持我们的社区教育合作伙伴。

如果您有我们可以分享的工作机会，或者您想咨询合作伙伴关系，可以通过Team@Honor.Software的电子邮件与我们联系。
